---
title: "klim open"
description: Install tools from a share token
---

Install tools from a compact share token generated by `klim share`.

## Usage

```bash
klim open <token> [flags]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<token>` | Share token (starts with `klim:v1:`) |

## Flags

| Flag | Short | Description |
|------|-------|-------------|
| `--yes` | `-y` | Install all tools without prompting |

## Examples

```bash
# Interactive — confirm before installing
klim open klim:v1:H4sIAAAA...

# Non-interactive — install all
klim open klim:v1:H4sIAAAA... --yes
```

## How It Works

1. Decodes the share token
2. Resolves tools against the local catalog
3. Installs each tool via the best available package manager for the current OS
